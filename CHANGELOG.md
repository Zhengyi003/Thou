# Changelog

## 1.0.0

- First public Thou companion boundary focused on OpenClaw connect, agent switch, session switch, history, and send.
- Public shell is narrowed to OpenClaw companion flow instead of dual-mode chat.
- Root repository license is now Apache-2.0.
- Alice page, settings, secrets, and runtime pipeline were removed from the app target; only Methodology A / B text assets remain in source.
- Repository root now includes the minimum GitHub release skeleton for an independent public repo.

## Known limits

- Tailnet remains the primary remote path.
- ATS is still broadly relaxed while the bridge remains on plaintext WebSocket.
- App Store submission assets and final review prep are still pending.
